This is a collection of useful pages for use within Cosmere RPG, the TTRPG developed by Brotherwise games based on the Stormlight Archive by Brandon Sanderson.

__Note__ it is important to realize that these pages will not be useful without a Stormlight Handbook (or maybe the Mistborn Handbook in the future) because required information is stored there. This is not meant to be a copy of the book and is not intending to break copywrite.


| Page Link | Description |
|-----------|-------------|
| [Character Creator](cosmere-character-creator.md) | This is the place to start when creating a character, and will help when making a character within the universe |
| [Radiant Paths](radiant-paths.md) | If you are interested in the magic system of Roshar this is a great place to look as well, while it can not be used until level 2 or later, a characters backstory could very relevant to their Radiant Path |
